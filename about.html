About the Developers!

  • First, thank you for playing our minesweeper app! It was truly a fun build that incorporated some of the knowledge gained in Justin's class.
  • Build: Angular.js, Bootstrap, fontawesome. This entire app is build on the simplest of platforms. It is only a small controller and local storage for a lite weight minesweeper app that only keeps local session storage.
  • The Angular.js - because angular is very good to use for a small app this minesweeper fit angular very well. The build was relatively easy and feels smooth for the player. Everything functions like the classic minesweeper without the bulk of the old windows machine.
  • The Bootstrap - used to make everything look nice and clean but also used to make our mobile players able to play.
  • What went wrong? Well, we had originally intended for this to be a game where you would log in and keep all your session variables on your mobile device or computer; however, we could only figure out the local storage. We also struggled with the ability to render all wins, loses, and the game board. The game board itself was bit tricky and didn't want to render for some states.
  • What went right? The game! Overall the game plays extremely well and smooth. Total success on the game board and the ability to play.
